How to Play
Installation:
Clone the repository or download the game files.
Ensure you have Node.js installed on your machine.
Navigate to the game directory and install dependencies:
npm install
Running the Game:
Start the server:
node server.js

Movement: W (up), A (left), S (down), D (right)
Select item: Press the number keys (1-9) to select item
Press E to open bag
Press M to open map
Press F to feed animal
Press T to tame animal
